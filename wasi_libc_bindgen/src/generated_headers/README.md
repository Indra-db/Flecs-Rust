# Generated Headers

This directory contains header files that are automatically generated during the build process.

## Files

- `bits/alltypes.h` - Generated from musl template files during build
  - Source templates: `src/libc-top-half/musl/arch/wasm32/bits/alltypes.h.in` and `src/libc-top-half/musl/include/alltypes.h.in`
  - Generated by: `build.rs` when building with `build-libc` feature
  - Contains conditional type definitions using musl's `__NEED_*` macro system

## Important Notes

- **Do not edit files in this directory manually** - they will be overwritten on the next build
- Files are automatically regenerated when:
  - Building with `build-libc` feature
  - Template files are modified
  - Build script is modified
